# solr-metal

`solr-metal` validates whether a cluster is healthy enough to trust at the bare-metal level: API reachability, node readiness, permissions, namespace lifecycle, DNS basics, storage binding, and core OpenShift health surfaces.

The CLI is `sm`.

Use the [user guide](user-guide.md) for command-level usage, the [architecture guide](architecture.md) for runtime design, and the [distribution guide](distribution.md) for release and delivery modes.
